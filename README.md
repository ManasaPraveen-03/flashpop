FlashPop – Smart Revision Web App

FlashPop is a lightweight web application designed to help students revise study material quickly and effectively. The app converts plain notes into flashcards and quizzes automatically, allowing users to practice active recall and track their learning progress.

The system includes user authentication, automatic flashcard generation from pasted notes, quiz practice with feedback, and progress tracking. FlashPop uses a simple web interface and stores data locally using JSON files, making it easy to run without installing a database.

Features

User signup and login

Paste notes and automatically generate flashcards

View and review flashcards

Quiz based on generated flashcards

Progress tracking for revision and quiz performance

Clean and vibrant web interface

File-based data storage (JSON) instead of a database
